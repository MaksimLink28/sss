Графічна програма

Для початку потрібно вирішити з якою бібліотекою ми будемо працювати Існують такі бібліотеки як:

  • tkinter
  • kivy
  • pyQt
  • turtle
Будемо ми використовувати tkinter а напишемо ми калькулятор

Для початку пропишуму стандартну структуру докумена на tkinter 

#iмпортуємо бібліотеку(завантажувати її не потрібно вона встроєна)
from tkinter import *
#вказуємо що змінна root е типу наше вікно
root = Tk()

#ключаємо програмі вічний цикл(щоб вона при запуску зразу не виключелась)
root.mainloop()

тепер давайте розберемо параметри екрану 

from tkinter import *
root = Tk()
#встановлюємо назву вікна
root.title("Hello World!")
#встановлюємо задній фон
root["bg"] = "green"
#встановлюємо ширину і довжину екрану
root.geometry("600x300")
#встановлюємо чи зможе наш користувач змінювати ширину чи довжину програми
root.resizable(width = False , height = False)

root.mainloop()

Тепер ми можемо приступити до розробки нашої програми Для того нам прийдеться розглянути декілька віджетів

Label

Віджет label представляє собою віджет-текст у якому ми просто виводимо текст у програму

from tkinter import *
root = Tk()
root.title("Hello World!")
root["bg"] = "green"
root.geometry("600x300")
root.resizable(width = False , height = False)
#У змінну lbl ми передаємо параметр label у якому ми передаємо ще один параметр text який відповідяє за текст у нашому label
lbl = Label(text = "Введіть перше число: ")
#це упаковщик нашої змінної label 
lbl.pack()

root.mainloop()

Entry

Змінна entry це поле для введення тексту Ось приклад її роботи 

from tkinter import *
root = Tk()
root.title("Hello World!")
root["bg"] = "green"
root.geometry("600x300")
root.resizable(width = False , height = False)

lbl = Label(text = "Введіть перше число: ")
lbl.pack()
#У змінній entr ми використовуємо параметр Entry який якраз і слугує нашим полем для введення тексту
entr = Entry()
#упаковуємо наше поле
entr.pack()


root.mainloop()

Button

Змінна Button це кнопка ось приклад її роботи 

from tkinter import *
root = Tk()
root.title("Hello World!")
root["bg"] = "green"
root.geometry("600x300")
root.resizable(width = False , height = False)

lbl = Label(text = "Введіть перше число: ")
lbl.pack()
entr = Entry()
entr.pack()

#Створюємо змінну plus у якій вказуємо що текст кнопки буде +
plus = Button(text = "+")
#Упаковуємо кнопку
plus.pack()

root.mainloop()

Але наша кнопка не робоча Давайте запрограмуємо кнопку щоб вона виводила Hello World у консоль 

from tkinter import *
root = Tk()
root.title("Hello World!")
root["bg"] = "green"
root.geometry("600x300")
root.resizable(width = False , height = False)
#у тій змінній якраз і буде працювати наша кнопка
def do():
	print("Hello World")

lbl = Label(text = "Введіть перше число: ")
lbl.pack()
entr = Entry()
entr.pack()

#Створюємо змінну plus у якій вказуємо що текст кнопки буде + далі створюємо змінну command у якій вказуєму через що буде працювати наша кнопка
plus = Button(text = "+", command = do)
#Упаковуємо кнопку
plus.pack()


root.mainloop()
